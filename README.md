# ROSA, temporary GoogleTest exercise
This repo will be removed later.

## Requirements
1. Build tool:
   * CMake version 3.10

## How to Build
### 1. Build the project:
```
mkdir build
cd build
cmake ../
make -j8
```

## How to Run
In `build` folder, run the executable as below to run the main code (which is essentially doing nothing.)
```
./RosaSimulator
```
and the following to run the tests
```
./hello_test
```
